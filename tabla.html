Tema 2: Elementos de Estructuración de una Página Web

  • Contenedor (wrapper): Un elemento envolvente que agrupa el contenido y lo centra en la página. Facilita el diseño de la página al proporcionar un ancho máximo y márgenes.
  • Cabecera (header): La parte superior de una página web que generalmente contiene el logotipo, el título y el menú principal.
  • Contenido (content): La sección principal de la página donde se muestra la información más relevante para los usuarios.
  • Menú (menu): Una lista de enlaces de navegación que permite a los usuarios moverse entre las diferentes secciones del sitio web.
  • Pie (footer): La parte inferior de la página que a menudo incluye información de contacto, enlaces a políticas del sitio y detalles adicionales.
  • Lateral (sidebar): Una columna lateral en la página que puede contener enlaces adicionales, widgets o contenido relacionado.
  • Etiqueta <div>: Un contenedor genérico que agrupa otros elementos y facilita el diseño de la página mediante el uso de CSS.

Tema 3: Estructuración Semántica de una Página Web

  • Etiqueta <header>: Define la cabecera de una página o una sección, incluyendo el logotipo, el título y el menú de navegación.
  • Etiqueta <nav>: Define una sección de navegación que contiene enlaces a otras páginas o secciones del sitio.
  • Etiqueta <main>: Define el contenido principal de la página, excluyendo el encabezado, el pie y las barras laterales.
  • Etiqueta <article>: Define un contenido independiente y autónomo, como un artículo de blog o una entrada de noticia.
  • Etiqueta <section>: Define una sección genérica en un documento, que puede contener uno o más artículos o contenido relacionado.
